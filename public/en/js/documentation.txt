a_          page specific area element

c_t_        color theme

j_link      page link, reroute to data-url attribute
j_G-        page specific group
j_i-        indicator label

j_dy-form   dynamic form (fill with JS drawSearch)
j_t-        table elements of JS importance

j_u-        underscore: input
t_          generic table elements
